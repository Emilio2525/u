Introducción a Python

Python es un lenguaje de programación de alto nivel, conocido por su simplicidad y legibilidad. Se utiliza en desarrollo web, análisis de datos, inteligencia artificial y más.

Características de Python

  • Sintaxis sencilla y clara.
  • Amplia biblioteca estándar.
  • Gran comunidad y soporte.
  • Compatible con múltiples plataformas.

Variables y Tipos de Datos

Las variables almacenan información. Python es un lenguaje de tipado dinámico, así que no necesitas declarar el tipo.

Tipos de Datos Comunes

  • Cadenas: Texto.
  • Enteros: Números enteros.
  • Flotantes: Números decimales.
  • Booleanos: Verdadero o falso.
  • Listas: Colecciones ordenadas y mutables.
  • Diccionarios: Colecciones de pares clave-valor.

Ejemplos

nombre = "Juan"
edad = 30
altura = 1.75
es_estudiante = True
lista = [1, 2, 3, 4]
diccionario = {"nombre": "Juan", "edad": 30}

Control de Flujo

Las estructuras de control permiten que tu programa tome decisiones y repita acciones.

Condicionales

if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Bucles

for i in range(5):
    print(i)

contador = 0
while contador < 5:
    print(contador)
    contador += 1

Funciones

Las funciones son bloques de código reutilizables. Puedes definir funciones y usar funciones integradas.

Definición de Funciones

def saludar(nombre):
    return f"Hola, {nombre}!"

print(saludar("Mundo"))

Funciones Lambda

suma = lambda a, b: a + b
print(suma(5, 3))  # Salida: 8

Clases y Objetos

La programación orientada a objetos permite crear estructuras complejas a través de clases y objetos.

class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
        return f"Hola, soy {self.nombre} y tengo {self.edad} años."

p = Persona("Ana", 25)
print(p.saludar())

Módulos y Paquetes

Python permite organizar el código en módulos y paquetes para mejorar la legibilidad y reutilización.

# Importando un módulo
import math

print(math.sqrt(16))  # Salida: 4.0

Manejo de Excepciones

El manejo de excepciones permite manejar errores en tiempo de ejecución de manera controlada.

try:
    resultado = 10 / 0
except ZeroDivisionError:
    print("No se puede dividir por cero.")
finally:
    print("Operación finalizada.")

Proyectos Prácticos

Proyecto 1: Juego de Adivinanza

import random

def juego_adivinanza():
    numero_secreto = random.randint(1, 100)
    intentos = 0
    adivinado = False

    while not adivinado:
        intento = int(input("Introduce tu número: "))
        intentos += 1
        if intento < numero_secreto:
            print("Demasiado bajo.")
        elif intento > numero_secreto:
            print("Demasiado alto.")
        else:
            adivinado = True
            print(f"¡Felicidades! Adivinaste en {intentos} intentos.")
juego_adivinanza()

Proyecto 2: Calculadora Simple

def calculadora():
    operacion = input("Introduce la operación (+, -, *, /): ")
    num1 = float(input("Introduce el primer número: "))
    num2 = float(input("Introduce el segundo número: "))
    
    if operacion == "+":
        print(f"Resultado: {num1 + num2}")
    elif operacion == "-":
        print(f"Resultado: {num1 - num2}")
    elif operacion == "*":
        print(f"Resultado: {num1 * num2}")
    elif operacion == "/":
        print(f"Resultado: {num1 / num2}")
    else:
        print("Operación no válida.")

calculadora()

Proyecto 3: Aplicación de Notas

notas = []

def agregar_nota(nota):
    notas.append(nota)

def mostrar_notas():
    for nota in notas:
        print(nota)

agregar_nota("Matemáticas: 90")
agregar_nota("Historia: 85")
mostrar_notas()

Ejercicios Prácticos

Ejercicio 1: Calcular el Área de un Círculo

Escribe un programa que pida el radio de un círculo y calcule su área.

import math

radio = float(input("Introduce el radio: "))
area = math.pi * (radio ** 2)
print(f"El área del círculo es: {area}")

Ejercicio 2: Contador de Vocales

Escribe un programa que cuente las vocales en una cadena de texto.

texto = input("Introduce un texto: ")
vocales = "aeiouAEIOU"
contador = sum(1 for letra in texto if letra in vocales)
print(f"Número de vocales: {contador}")
